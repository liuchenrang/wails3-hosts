专为开发者打造，告别繁琐的 hosts 管理

分组管理

按项目或环境创建 hosts 分组，支持一键启用/禁用整组，多环境切换轻松自如。

条目管理

可视化添加、编辑、删除 hosts 条目，填写 IP + 主机名 + 注释，告别手动敲命令。

快捷键操作

Cmd+S (Mac) / Ctrl+S (Win) 快速应用配置，效率翻倍。

版本历史

自动保存每次修改的历史版本，误操作一键回滚，再也不怕改错配置。

多语言支持

明暗主题

支持明亮和暗色两种主题，深夜 coding 也不刺眼，护眼舒适。
