# 提案执行摘要

## 📋 变更ID
`add-windows-support`

## 🎯 目标
为 Hosts Manager 添加完整的 Windows 平台支持，实现跨平台一致性。

## 💡 核心方案

### 架构模式
**策略模式** + **依赖注入**
```
PrivilegeElevator (接口)
    ├── UnixElevator (sudo)
    └── WindowsElevator (UAC)
```

### 关键技术
- **Unix**: 使用 sudo 命令（保持现有实现）
- **Windows**: 使用 UAC (User Account Control) 提权
- **平台检测**: `runtime.GOOS` + 构建标签
- **Windows API**: `golang.org/x/sys/windows`

## 📦 交付物

### 新增组件 (4个文件)
1. `privilege.go` - 权限提升器接口
2. `privilege_unix.go` - Unix 实现
3. `privilege_windows.go` - Windows 实现  
4. `privilege_windows_test.go` - 单元测试

### 修改组件 (3个文件)
1. `hosts_file_operator.go` - 使用接口注入
2. `main.go` - 工厂函数修改
3. `hosts_app_service.go` - 错误处理调整
