## 环境准备

### 硬件和软件环境
- [ ] Windows 10 (版本 1607+) 或 Windows 11
- [ ] Go 1.24+ 已安装
- [ ] Node.js 18+ 已安装
- [ ] 应用已成功编译（wails3-hosts.exe）
- [ ] 当前使用标准用户账户（非管理员）

### 前置检查
- [ ] UAC 已启用
- [ ] hosts 文件路径: `C:\Windows\System32\drivers\etc\hosts`
- [ ] 配置目录: `%APPDATA%\hosts-manager`
- [ ] 备份目录: `%APPDATA%\hosts-manager\backups`
